## Local Development

#### Step-1

clone this repo

```sh
git clone https://github.com/Swoyam1/beam-dynamics-frontend.git
```

#### Step-2

Install all dependencies

```sh
npm install
```

#### Step-3: Starting the  application

To Run the application in development mode execute this script

```sh
npm run dev
```
